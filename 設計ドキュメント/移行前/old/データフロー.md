# データフロー（移行前）

移行前（暫定構成）のデータフローを示すシーケンス図です。

```mermaid
sequenceDiagram
  autonumber
  %% --- 登場人物（データ/システム）の定義 ---
  box "外部環境 (Databricks)" #f0f0f0
    participant DBX as Databricks<br/>(外部クラスタ)
  end

  box "AWS環境" #e1f5fe
    participant RDS as RDS<br/>(Northwind)
    participant Bronze as Bronze<br/>(S3 Delta)
    participant Silver as Silver<br/>(S3 Delta)
    participant Gold as Gold<br/>(S3 Delta)
  end

  box "Ops & Governance" #fff3e0
    participant Ops as Ops<br/>(Logs/DQ)
  end

  %% --- 処理フロー ---
  Note over DBX, Gold: 日次バッチ処理開始

  %% 1. Ingest (インターネット経由)
  DBX->>RDS: 1. JDBC接続 (インターネット経由/SSL)
  RDS-->>DBX: データ返却
  DBX->>Bronze: 2. データ保存 (S3 API + Access Key)
  activate Bronze
  Bronze->>Ops: 3. ログ記録 (Ingestion Log)
  deactivate Bronze

  %% 2. Transform
  Bronze->>Silver: 4. クレンジング・標準化
  activate Silver
  Silver->>Ops: 5. 品質チェック (DQ Check)
  
  opt 品質チェックNGの場合
      Ops-->>Silver: 処理中断 / アラート通知
  end
  deactivate Silver

  %% 3. Aggregate
  Silver->>Gold: 6. 集計・マート作成
  activate Gold
  Gold-->>Ops: 7. 完了記録
  deactivate Gold
```

## 移行前の注意点

1. **JDBC接続はインターネット経由**: RDSのパブリックエンドポイントを使用
2. **SSL必須**: `sslmode=require` を接続文字列に含める
3. **S3接続はAccess Key**: IAM Role (Instance Profile) は使えないため、Databricks Secrets に Access Key を保存
