# ダッシュボード設計書

## 概要

| 項目 | 内容 |
| --- | --- |
| 対象 Epic | Epic 5: BI ダッシュボードとレポート自動配信 |
| 対象 Story | 5-1（売上ダッシュボード）、5-2（在庫状況ダッシュボード） |
| 目的 | Gold 層の売上集計データおよび Silver 層の在庫データを Databricks SQL ダッシュボードで可視化する |
| プラットフォーム | Databricks SQL ダッシュボード（新 UI。2026-03-05 に Legacy ダッシュボード廃止済み） |
| コンピューティング | SQL Warehouse（推奨: `northwind_warehouse`, サイズ `2X-Small`, 自動停止 10 分） |

---

## ダッシュボード一覧

| ダッシュボード名 | 対応 Story | 使用データ層 | 目的 |
| --- | --- | --- | --- |
| Northwind 売上ダッシュボード | 5-1 | Gold | 売上トレンド・商品別ランキング・顧客別売上の可視化 |
| Northwind 在庫状況ダッシュボード | 5-2 | Silver | 在庫切れリスク・発注点割れアラートの可視化 |

## 1. Northwind 売上ダッシュボード（Story 5-1）

### 1-1. 使用テーブル

| テーブル | 層 | 用途 |
| --- | --- | --- |
| `northwind_catalog.gold.order_summary` | Gold | 日次注文サマリ → 月別売上推移 |

## 2. Northwind 在庫状況ダッシュボード（Story 5-2）

### 2-1. 使用テーブル

| テーブル | 層 | 用途 |
| --- | --- | --- |
| `northwind_catalog.silver.products` | Silver | 在庫情報（在庫数・発注点・廃止フラグ） |
| `northwind_catalog.bronze.categories` | Bronze | カテゴリ名（JOIN） |
| `northwind_catalog.bronze.suppliers` | Bronze | 仕入先名（発注点割れアラートで JOIN） |

### 2-2. ウィジェット構成

| # | ウィジェット名 | データセット | 可視化タイプ | 配置 |
| --- | --- | --- | --- | --- |
| 1 | カテゴリ別在庫リスク件数 | `カテゴリ別在庫サマリ` | 縦棒グラフ（Bar） | 上段左（幅 1/2） |
| 2 | 在庫切れリスク商品 | `在庫切れリスク商品` | 横棒グラフ（Bar / Horizontal） | 上段右（幅 1/2） |
| 3 | 発注点割れアラート | `発注点割れアラート` | テーブル（Table） | 中段（全幅） |
| 4 | 在庫状況一覧 | `在庫状況一覧` | テーブル（Table） | 下段（全幅） |

### 2-3. データセット定義

#### データセット: 在庫状況一覧

- **クエリファイル**: `notebooks_sprint4/story5-2/5-2_在庫状況一覧.sql`
- **ソーステーブル**: `silver.products` LEFT JOIN `bronze.categories`

| 出力列 | 型 | 説明 |
| --- | --- | --- |
| `product_id` | INT | 商品 ID |
| `product_name` | STRING | 商品名 |
| `category_name` | STRING | カテゴリ名 |
| `unit_price` | DECIMAL | 単価 |
| `units_in_stock` | INT | 現在在庫数 |
| `reorder_level` | INT | 発注点 |
| `units_on_order` | INT | 発注中数量 |
| `discontinued` | INT | 廃止フラグ（1: 廃止） |
| `stock_status` | STRING | リスク判定（下表参照） |

- **リスク判定ロジック（`stock_status`）**

| 条件 | 判定値 | 優先度 |
| --- | --- | --- |
| `discontinued = 1` | `廃止` | 4（最低） |
| `units_in_stock = 0` | `在庫切れ` | 1（最高） |
| `units_in_stock <= reorder_level AND units_on_order = 0` | `発注点割れ（未発注）` | 2 |
| `units_in_stock <= reorder_level AND units_on_order > 0` | `発注点割れ（発注済）` | 3 |
| その他 | `正常` | 5 |

- **可視化設定**

| 設定項目 | 値 |
| --- | --- |
| タイプ | テーブル（Table） |
| 条件付き書式 | `stock_status` 列: 在庫切れ→赤、発注点割れ→黄、正常→緑 |

---

#### データセット: 在庫切れリスク商品

- **クエリファイル**: `notebooks_sprint4/story5-2/5-2_在庫切れリスク商品.sql`
- **ソーステーブル**: `silver.products` LEFT JOIN `bronze.categories`
- **フィルタ**: `discontinued = 0` かつ `units_in_stock <= reorder_level`

| 出力列 | 型 | 説明 |
| --- | --- | --- |
| `product_name` | STRING | 商品名 |
| `category_name` | STRING | カテゴリ名 |
| `units_in_stock` | INT | 現在在庫数 |
| `reorder_level` | INT | 発注点 |
| `units_on_order` | INT | 発注中数量 |
| `unit_price` | DECIMAL | 単価 |
| `alert_level` | STRING | アラートレベル（在庫切れ / 要発注 / 発注済（入荷待ち）） |

- **可視化設定**

| 設定項目 | 値 |
| --- | --- |
| タイプ | 横棒グラフ（Bar / Horizontal） |
| Y 軸 | `product_name` |
| X 軸 | `units_in_stock`、`reorder_level`（重ねて表示） |

---

#### データセット: カテゴリ別在庫サマリ

- **クエリファイル**: `notebooks_sprint4/story5-2/5-2_カテゴリ別在庫サマリ.sql`
- **ソーステーブル**: `silver.products` LEFT JOIN `bronze.categories`

| 出力列 | 型 | 説明 |
| --- | --- | --- |
| `category_name` | STRING | カテゴリ名 |
| `product_count` | BIGINT | カテゴリ内商品数 |
| `out_of_stock_count` | BIGINT | 在庫切れ件数（非廃止） |
| `below_reorder_count` | BIGINT | 発注点割れ件数（在庫 > 0、非廃止） |
| `total_stock` | BIGINT | カテゴリ内在庫数合計 |
| `stock_value` | DECIMAL | カテゴリ内在庫金額合計（units_in_stock × unit_price） |

- **可視化設定**

| 設定項目 | 値 |
| --- | --- |
| タイプ | 縦棒グラフ（Bar） |
| X 軸 | `category_name` |
| Y 軸 | `out_of_stock_count`、`below_reorder_count`（積み上げ） |
| タイトル | カテゴリ別在庫リスク件数 |
| ソート | `out_of_stock_count` 降順 |

---

#### データセット: 発注点割れアラート

- **クエリファイル**: `notebooks_sprint4/story5-2/5-2_発注点割れアラート.sql`
- **ソーステーブル**: `silver.products` LEFT JOIN `bronze.categories` LEFT JOIN `bronze.suppliers`
- **フィルタ**: `discontinued = 0` かつ `units_in_stock < reorder_level`

| 出力列 | 型 | 説明 |
| --- | --- | --- |
| `product_name` | STRING | 商品名 |
| `category_name` | STRING | カテゴリ名 |
| `supplier_name` | STRING | 仕入先企業名 |
| `units_in_stock` | INT | 現在在庫数 |
| `reorder_level` | INT | 発注点 |
| `shortage_qty` | INT | 不足数量（reorder_level - units_in_stock） |
| `units_on_order` | INT | 発注中数量 |
| `unit_price` | DECIMAL | 単価 |
| `estimated_order_cost` | DECIMAL | 推定発注金額（shortage_qty × unit_price） |
| `order_status` | STRING | 発注状況（発注済 / 未発注 - 要アクション） |

- **可視化設定**

| 設定項目 | 値 |
| --- | --- |
| タイプ | テーブル（Table） |
| 条件付き書式 | `order_status = '未発注 - 要アクション'` の行を赤色ハイライト |
| ソート | `shortage_qty` 降順 |
