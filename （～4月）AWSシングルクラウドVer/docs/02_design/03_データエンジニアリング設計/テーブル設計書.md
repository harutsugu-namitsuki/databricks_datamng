## 1. Bronze層

### 1.1 概要
- **保存形式**: Delta Lake
- **保存場所**: `s3://lake-northwind-<account-id>/bronze/`

### 1.2 テーブル一覧

| テーブル名 | 説明 | 件数 |
|-----------|------|------|
| bronze.categories | カテゴリマスタ（Raw）。商品を分類するカテゴリ情報。RDS PostgreSQLから日次取り込み。 | 8 |
| bronze.customers | 顧客マスタ（Raw）。取引先企業の基本情報（社名・担当者・住所）。担当者氏名・連絡先等の個人情報を含む。RDSから日次取り込み。 | 91 |
| bronze.employees | 従業員マスタ（Raw）。Northwind社の従業員情報（氏名・役職・連絡先）。個人情報を含む。RDSから日次取り込み。 | 9 |
| bronze.suppliers | 仕入先マスタ（Raw）。商品の仕入先企業情報。RDSから日次取り込み。 | 29 |
| bronze.shippers | 配送業者マスタ（Raw）。注文の配送を担当する業者情報。RDSから日次取り込み。 | 3 |
| bronze.products | 商品マスタ（Raw）。販売商品の基本情報（商品名・価格・在庫数）。RDSから日次取り込み。 | 77 |
| bronze.orders | 受注データ（Raw）。顧客からの注文ヘッダ情報（注文日・配送先・運賃）。約830件。RDSから日次取り込み。 | 830 |
| bronze.order_details | 受注明細データ（Raw）。注文ごとの商品明細（商品ID・単価・数量・割引）。約2,155件。RDSから日次取り込み。 | 2,155 |
| bronze.region | 地域マスタ（Raw）。米国の地域区分（Eastern, Western等）。RDSから日次取り込み。 | 4 |
| bronze.territories | テリトリマスタ（Raw）。営業担当エリアの定義。地域（region）に紐づく。RDSから日次取り込み。 | 53 |
| bronze.us_states | 米国州マスタ（Raw）。米国の州名・略称・地域区分。参照用マスタ。RDSから日次取り込み。 | 51 |
| bronze.employee_territories | 従業員テリトリ関連（Raw）。従業員とテリトリの多対多関連テーブル。RDSから日次取り込み。 | 49 |
| bronze.customer_demographics | 顧客デモグラフィクス（Raw）。顧客の属性分類定義。現在データなし。RDSから日次取り込み。 | 0 |
| bronze.customer_customer_demo | 顧客デモ関連（Raw）。顧客とデモグラフィクスの関連テーブル。現在データなし。RDSから日次取り込み。 | 0 |

### 1.3 追加カラム（メタデータ）

|---------|-----|------|

## 2. Silver層

### 2.1 概要
- **保存形式**: Delta Lake
- **保存場所**: `s3://lake-northwind-<account-id>/silver/`

### 2.2 テーブル一覧

| テーブル名 | ソース | 変換内容 |
|-----------|--------|---------|
| silver.customers | bronze.customers | trim, upper(country), Null→デフォルト値 |
| silver.orders | bronze.orders | 日付型cast(date), Null→0, upper(ship_country) |
| silver.order_details | bronze.order_details | decimal cast, line_total計算 |
| silver.products | bronze.products | trim, decimal cast(unit_price), Null→0 |

### 2.3 クレンジングルール

| ルール | 説明 |
|--------|------|
| Null→デフォルト値 | 数値は0、文字列は空文字 |
| 日付型変換 | VARCHAR→DATE/TIMESTAMP |
| 正規化 | 文字列のトリム、大文字統一 |
