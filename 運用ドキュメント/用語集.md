# 用語集

## A-Z

| 用語 | 説明 |
|------|------|
| **ADLS Gen2** | Azure Data Lake Storage Gen2。階層型名前空間を持つAzureのオブジェクトストレージ。 |
| **Access Connector** | Azure Databricks用のAccess Connector。Managed Identityを使用してADLSにアクセス。 |
| **Bronze層** | メダリオンアーキテクチャの最下層。ソースデータをそのまま保存（Raw Data）。 |
| **Delta Lake** | Sparkエコシステムのオープンソースストレージフォーマット。ACID対応。 |
| **ETL** | Extract-Transform-Load。データ抽出・変換・読み込みの処理パイプライン。 |
| **External Location** | Unity Catalogで外部ストレージを参照するための設定。 |
| **Gold層** | メダリオンアーキテクチャの最上層。集計・マートデータ。BIツールから参照。 |
| **IAM** | Identity and Access Management。AWS/Azureの認証・認可サービス。 |
| **JDBC** | Java Database Connectivity。データベース接続の標準API。 |
| **Managed Identity** | Azureのパスワードレス認証方式。自動ローテーション。 |
| **Medallion Architecture** | Bronze/Silver/Goldの3層でデータを管理するアーキテクチャ。 |
| **Northwind** | Microsoftが提供するサンプルデータベース。顧客・注文・商品データ。 |
| **RDS** | Amazon Relational Database Service。AWSのマネージドRDBサービス。 |
| **Silver層** | メダリオンアーキテクチャの中間層。クレンジング・標準化されたデータ。 |
| **Storage Credential** | Unity CatalogでストレージへのアクセスCredentialを管理する機能。 |
| **Unity Catalog** | Databricksのデータガバナンス機能。メタデータ管理、リネージ、アクセス制御。 |

## 日本語

| 用語 | 説明 |
|------|------|
| **クレンジング** | データの欠損値処理、重複除去、形式統一などの品質向上処理。 |
| **リネージ** | データの出自と変換履歴を追跡する機能。 |
| **マート** | 特定の分析目的に最適化されたデータセット（Gold層）。 |
| **メタデータ** | データの説明情報（テーブル名、カラム名、型、説明など）。 |
