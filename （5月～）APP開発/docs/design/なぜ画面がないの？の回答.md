質問への回答
今のStreamlitには HTMLファイルは一切ありません。 StreamlitはPythonを書くと内部でHTMLを自動生成してくれる仕組みで、HTMLを直接書く必要がないのが特徴です。ただし、UIのカスタマイズの自由度が低く、「普通のWebアプリ」とは別物です。

作ったもの（HTML/CSS/JS + FastAPI 版）
構成

ブラウザ (HTML + CSS + JavaScript)  ← これが「画面」
        ↕ HTTP通信 (fetch API)
FastAPI (Python バックエンド)        ← データを返すサーバー
        ↕ psycopg2
RDS PostgreSQL
HTMLファイル（実際の画面）
ファイル	画面
web/admin/login.html	業務管理 ログイン
web/admin/dashboard.html	KPI・最近の注文・在庫アラート
web/admin/products.html	商品CRUD
web/admin/inventory.html	在庫調整
web/admin/employees.html	従業員管理
web/store/login.html	ストア ログイン
web/store/catalog.html	商品一覧（カード形式）
web/store/cart.html	カート
web/store/checkout.html	注文確定
web/store/history.html	注文履歴
起動方法（RDSに繋がる環境なら動く）

cd APP開発
cp .env.example .env   # RDS接続情報を書く

cd src
uvicorn api.main:app --reload --port 8000
ブラウザで http://localhost:8000/static/store/login.html を開けば画面が出ます。
