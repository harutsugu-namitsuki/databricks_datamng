# 権限設計（AWSシングルクラウド版）

## 1. 設計方針

**最小権限の原則**を適用し、各レイヤーごとにロールベースのアクセス制御を行う。

---

## 2. AWS IAM 権限

### 2.1 IAM Role 一覧

| IAM Role | 用途 | 主な権限 |
|----------|------|---------|
| `databricks-workspace-role` | Databricks ワークスペース用 | EC2, VPC, S3 基本操作 |
| `databricks-unity-catalog-role` | Unity Catalog 用 | S3 Read/Write（lake-northwind） |
| `databricks-instance-profile` | クラスタ用 Instance Profile | S3 Read/Write, Secrets Manager Read |

### 2.2 S3 バケットポリシー

| 対象 | 許可アクション | 許可 Principal |
|------|--------------|---------------|
| `s3://lake-northwind/*` | GetObject, PutObject, DeleteObject, ListBucket | databricks-unity-catalog-role |

### 2.3 Secrets Manager

| 対象 | 許可アクション | 許可 Principal |
|------|--------------|---------------|
| `northwind/rds/*` | GetSecretValue | databricks-instance-profile |

## 3. Unity Catalog 権限

### 3.1 ロール定義

| ロール | 対象者 | 説明 |
|--------|-------|------|
| **Admin** | プロジェクト管理者 | 全権限（カタログ/スキーマ/テーブルの管理） |
| **DataEngineer** | ETL開発者 | Bronze/Silver/Gold/Ops の読み書き |
| **Analyst** | データ分析者 | Gold の読み取りのみ |

### 3.2 スキーマ別アクセスマトリクス

| スキーマ | Admin | DataEngineer | Analyst |
|---------|-------|-------------|---------|
| bronze | ✅ ALL | ✅ ALL | ❌ なし |
| silver | ✅ ALL | ✅ ALL | ❌ なし |
| gold | ✅ ALL | ✅ ALL | ✅ SELECT |
| ops | ✅ ALL | ✅ ALL | ✅ SELECT |

### 3.3 アクセス制限の理由

| 制限 | 理由 |
|------|------|
| Bronze: Analyst不可 | 生データには個人情報やノイズが含まれる可能性 |
| Silver: Analyst不可 | 中間処理データへの直接アクセスは誤解を招く |
| Gold: Analyst読み取り | 提供用に整形されたデータのみ公開 |
| 必要に応じて | Gold上にマスキングViewを作成して公開 |

### 3.4 権限付与SQL

```sql
-- Admin
GRANT ALL PRIVILEGES ON CATALOG northwind_catalog TO admin_group;

-- DataEngineer
GRANT ALL PRIVILEGES ON SCHEMA northwind_catalog.bronze TO engineer_group;
GRANT ALL PRIVILEGES ON SCHEMA northwind_catalog.silver TO engineer_group;
GRANT ALL PRIVILEGES ON SCHEMA northwind_catalog.gold TO engineer_group;
GRANT ALL PRIVILEGES ON SCHEMA northwind_catalog.ops TO engineer_group;

-- Analyst
GRANT USAGE ON CATALOG northwind_catalog TO analyst_group;
GRANT USAGE ON SCHEMA northwind_catalog.gold TO analyst_group;
GRANT SELECT ON SCHEMA northwind_catalog.gold TO analyst_group;
GRANT USAGE ON SCHEMA northwind_catalog.ops TO analyst_group;
GRANT SELECT ON SCHEMA northwind_catalog.ops TO analyst_group;
```
