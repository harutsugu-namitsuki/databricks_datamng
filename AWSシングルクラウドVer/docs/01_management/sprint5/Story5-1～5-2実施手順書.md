## Phase 1: SQL Warehouse の準備

### 目的

Databricks SQL ダッシュボードを利用するには SQL Warehouse（SQL コンピューティングリソース）が必要となる。既存の SQL Warehouse がない場合は新規作成する。

### Step 1-1: SQL Warehouse の確認・作成

1. Databricks UI 左メニューから **「SQL ウェアハウス」（SQL Warehouses）** をクリック
2. 既存の SQL Warehouse が **Running** または **Stopped** 状態で存在する場合はそれを使用する → Step 1-2 へ
3. 存在しない場合は **「SQL ウェアハウスを作成」（Create SQL warehouse）** をクリック

| 設定項目 | 推奨値 | 説明 |
|---------|--------|------|
| 名前 | `northwind_warehouse` | 任意の名前 |
| クラスターサイズ | `2X-Small` | 学習用途のため最小サイズで十分 |
| 自動停止 | `10 分` | コスト削減のため短めに設定 |
| タイプ | `Serverless`（利用可能な場合） | 起動が速く従量課金 |

4. **「作成」（Create）** をクリック

### Step 1-2: SQL Warehouse の起動確認

1. 作成した SQL Warehouse の状態が **Running** であることを確認
2. 停止中の場合は **「開始」（Start）** をクリックし、起動を待つ

> **コスト注意**: SQL Warehouse は起動中に課金される。作業が終わったら停止すること。`自動停止` を設定しておけば、指定時間アイドル後に自動で停止する。

| データセット名 | クエリファイル |
