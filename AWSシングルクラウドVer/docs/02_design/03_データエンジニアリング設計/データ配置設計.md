```
s3://lake-northwind-<account-id>/
├── bronze/                    ← _load_date でパーティション
│   ├── categories/
│   ├── customers/
│   ├── employees/
│   ├── suppliers/
│   ├── shippers/
│   ├── products/
│   ├── orders/
│   ├── order_details/
│   ├── region/
│   ├── territories/
│   ├── us_states/
│   ├── employee_territories/
│   ├── customer_demographics/
│   └── customer_customer_demo/
├── silver/
│   ├── customers/
│   ├── orders/
│   ├── order_details/
│   └── products/
├── gold/
│   ├── sales_by_product/
│   ├── sales_by_customer/
│   ├── sales_by_category/
│   └── order_summary/
└── ops/
    ├── job_runs/
    ├── ingestion_log/
    └── dq_results/
```

---

## 2. 論理スキーマ（Databricks上）

| スキーマ | 目的 | 更新方式 |
|---------|------|---------|
| **bronze** | 取り込み生データ（スナップショット/増分） | Append |
| **silver** | 型・コード・重複・NULL等を整えた標準層 | Merge/Overwrite |
| **gold** | 提供用（ファクト/ディメンション/集計） | Overwrite |
| **ops** | 運用ログ、品質結果 | Append |

---

## 3. 共通メタ列（全Bronzeテーブルに付与）

| カラム名 | 型 | 説明 | 例 |
|---------|-----|------|-----|
| `_run_id` | STRING | ジョブ実行ID（UUID） | `550e8400-e29b-41d4-a716-...` |
| `_load_date` | DATE | 取り込み基準日 | `2026-03-08` |
| `_ingest_ts` | TIMESTAMP | 取り込みタイムスタンプ | `2026-03-08T07:00:00Z` |
| `_source_system` | STRING | ソースシステム名 | `rds_northwind` |

---

## 4. S3パス命名規則

```
s3://lake-northwind-<account-id>/<layer>/<table_name>/
```

| 項目 | 命名規則 |
|------|---------|
| バケット名 | `lake-northwind-<account-id>` |
| レイヤー | `bronze` / `silver` / `gold` / `ops` |
| テーブル名 | スネークケース（小文字_区切り） |

### パス例

| レイヤー | パス |
|---------|------|
| Bronze | `s3://lake-northwind-<account-id>/bronze/customers/` |
| Silver | `s3://lake-northwind-<account-id>/silver/customers/` |
| Gold | `s3://lake-northwind-<account-id>/gold/sales_by_product/` |
| Ops | `s3://lake-northwind-<account-id>/ops/job_runs/` |

---

## 5. パーティション戦略

| レイヤー | パーティションキー | 備考 |
|---------|-----------------|------|
| Bronze | `_load_date` | 日次スナップショットの管理 |
| Silver | なし（テーブルサイズが小さいため） | 必要に応じて追加 |
| Gold | なし（テーブルサイズが小さいため） | 必要に応じて追加 |
| Ops | なし | Append方式で蓄積 |
